About The Green Space Place!

Who are we?

Hello! Our names are Amy and Kaylee and we students at the University of Sheffield studying Neuroscience and Geography. The production of this website has been instigated by the Code First Girls: Coding course. Our aim was to produce a webapp which would help locate greenspaces closest to the user!

Who are Code First?

Code First: Girls is an organisation which works with companies to encourage woman to pursue careers in technology. We were both part of a course which began to touch upon the use of python (a type of coding language) for the use of website design. Find out some mroe information here: https://www.codefirstgirls.org.uk/
